# Kattbutik React

En kattbutik byggd med React, React Router och React Bootstrap.

## Funktioner

- Startsida
- Om mig-sida
- Lista över kattraser från The Cat API
- Pagination för att bläddra mellan katter

## Tekniker

- React
- React Router
- React Bootstrap
- Vite

## Installation

Klona projektet:

```bash
git clone https://github.com/themyzteziz/kattbutik-react.git

Installera dependencies:

npm install

Starta projektet:

npm run dev
